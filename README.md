## 🎯 Features

* 📈 **Trend Analysis**
  Compare current vs previous performance cycle

* ⚠️ **OrgPulse (Risk Detection)**
  Detect burnout / performance drop
  Example: -13 tasks → alert

* 🤖 **AI Appraisal Generator**
  Reduces review time from **2h 15m → 28m**

* 🔍 **WHY Engine (Explainability)**
  Transparent scoring using weighted signals:
  Tasks (30%) | PRs (25%) | Attendance (20%) | Docs (15%)

* ⚖️ **DecisionGuard (Bias Detection)**
  Ensures fair, data-driven evaluation

## ⚙️ How It Works

1. Collects data from HR systems + work tools (GitHub, Jira)
2. Converts activity into performance signals
3. AI generates appraisal summary
4. DecisionGuard checks for bias
5. WHY Engine explains final score
